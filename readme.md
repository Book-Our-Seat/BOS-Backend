### SETUP

Install and Run ```postgresql```  
1. ```brew install postresql@15```
2. ```brew services start postgresql```




### Run
1. Run ```npm install ```
2. Run ```npm run dev```



#### Packages
Token authentication ```json-web-token```
Cron : ```node-cron```


#### Seed the database
Run ```node seeders/seed.js``` at the root of the project
